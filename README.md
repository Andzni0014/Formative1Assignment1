Introduction

Social Sparks is a program which was created to help Cora with maintaining a balance of her personal and social life. This app acts as a reminder by giving suggestions on what to do based on the time of day. It’s beneficial to the user because it simplifies their day and makes socializing easier for those who do not find it easy to communicate/socialize
Overall Purpose
•	Takes user input (times of day)
•	Processes it
•	Displays a suggested message based on that time
•	Includes a clear/rest button

Core Function
•	Takes a time-related keyword from the user 
•	Uses if-else conditions to match keywords 
•	Displays a social suggestion message 
•	Includes debug logging and reset functionality 

Social spark Suggestions/Logic:
morning: Send a 'Good Morning' message to your friends and family.
mid-day: Share something interesting with your friends and family.
afternoon: Send a 'Thinking of you' message to your spouse.
snack time: Call a loved one for 5-min to catch up
dinner: Leave a thoughtful message on a loved ones post
after dinner: Leave a thoughtful message on a loved one
night: Leave a 'Good night' message to your loved ones
